# Spring-Shop
   Java project with crud operations on web.
   
 ## Prerequisites
 * Java 1.8
 * Maven >=3.3.9
 * MongoDB 3.4.3
 * Bower
    - sudo apt-get update
    - sudo apt-get install nodejs
    - sudo apt-get install npm
    - sudo npm install -g bower 
    - sudo npm install -g bowerrc
 
 
 ## Deployment
 To run locally execute
 
      bower install
      
      mvn package
      java -jar target/*.jar
